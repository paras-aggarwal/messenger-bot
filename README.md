# Messenger BOT

A BOT is a computer program that create a conversation with a person over a medium via textual methods

### Technology Stack

| Name | Description |
| ------ | ------ |
| Node.Js | Backend Server |
| Express | Routing |
| Facebook | Medium |
| Dialog Flow | Response collector |
